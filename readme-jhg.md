# 令和最新車轆版諺文

Yes, we reinvented the wheels again.

諺文，車轆界最愛之一。諺文確實係方塊字表音最佳選擇，咁呢個車轆同其他車轆有咩區別呢？

無，其實就係全面支持各地粵語方言（理論上元音音位不多者皆可，故閩南話、客家話、官話等皆可），音位規劃儘量遵循諺文本身（部分實在無Unicode就用其他將就）。造成有個缺點就係，用先好撚多非標諺文等等。反正都唔會實際用，是但啦。

## 一覽

### 初聲

<img src="pic\jhg1.png"/>

青色者於多數環境不能正常組合。紅色者爲後備用。（知組Unicode無入，僅爲替代字符，tj實爲「ᄅᄐ」）

- `bb`、`dd`、`gg`多以內爆音出現，「gg」未見過後備用。不論清濁皆可變內爆音。詳細可見：[[FoLiSC-3\]粤方言 “帮端母内爆音出自侗台语底层”说再探讨——以桂东南勾漏片方言有音无字词和送气分调为切入点 - 知乎](https://zhuanlan.zhihu.com/p/646908459)
- `mb`、`nd`、`ngg`在粵語多數來自對應鼻音，應無同原來鼻音對立，後備用。
- `v`同`w`粵語應無對立，後備用。
- 粵語翹只有一組，翹者加圏即可，左歪右歪難分。`sl-sh`可以用「ᄰ-ᄉ」。
- 「ᅙ」僅在細音存在零聲母同以母對立時使用，如「拎（白讀｜ing1｜ᅙᅵᆼ）」同「英（jing1｜잉）」。
- `j`，`gw/kw｜gv/kv`個`w｜v`等聲母之視作介音由中聲實現，見下。

### 中聲

<img src="pic\jhg2.png"/>

青色者於多數環境不能正常組合。紅色者並未有實際用途。星號字處理較爲特殊，請仔細閱讀下方說明。

- 保證基本每個韻腹可以有齊介音（即「i」同「u」），除非不合法。 
- 「ᅣ 」「ᆦ 」「ᆘ 」「ힽ」「ᆙ」：
  - 用例：
    - 【1.  未融合，無單獨e（即ia｜ie），如新會羅坑】「ᅣ」`ja/ia`，「 ᅸ」`wia`
    - 【2.  已融合，如廣州話】「ᅣ」`e`，「 ᅸ」`we`，「ᆙ」`je`｜「ᆘ 」`jaa`
    - 【3.  過渡，如石歧話】`e`，「 ᅸ」`we`，「ᆙ」`je`｜「ᆘ 」`ja/ia`，「ힽ」`wia`
  - 「ᆘ 」：【2.】【3.】用，如「也（jaa5｜ᄋᆘ〯ﾞ ）」「喫（白讀｜jaak3｜ᄋᆘᆨ〮）」等。
  - 「ᆙ」：【2.】【3.】用，如「野（je5｜ᄋᆙ〯ﾞ）」「膉（白讀｜jek1｜ᄋᆙᆨ）」等。
  - 「ힽ」：【3.】用，由於Unicode無「ᅩ」同「ᆘ」個組合，只能用「ힽ」代替。經驗證有方言有此音位，故保留。
- 「ᆞ」：此音位正常不單獨出現，單獨出現時多數爲語氣詞或口語字（方言），如「就（白讀｜a6｜ᄋᆞ〮ﾞ）」
- 「ᆡ 」「ᆈ」：此音位僅方言`eoy`、`oi`、`ooi`有對立時，作爲`eoy`使用，如「趣（ceoy3｜ᄎᆡ〮）」「菜（coi3｜체〮）」「脆（cooi3｜최〮）」，其他情況下`eoy`寫作「ᅬ」即可。至於`jooi`同`jeoy`音兩者互補，粵語應無此種對立，故毋須再立一字。
- 「ᆝ」：`ja`除`jai`以外同`jaa`互補。除非不互補否則毋須使用此字母。
- 「ᆢ」：師韻，如「思（sy1｜ᄉᆢ）」
- 「ᅩ」：以下爲方言之中需要此音位之情況：
  - `ou`，除非此`o`爲/ɔ/。
  - 效攝發生元音融合但音位仍然存在：
    - 以聲母保持獨立，孤攝等（如「孤」「箍」）等爲圓脣化・脣齒化（如`gwu/kwu`或`gvu/kvu`），而效攝則爲`gu/ku`，作爲`gu/ku`之`u`使用。
    - 融合後在`u`同`o`之間，即`oo`。
  - 宕攝開口（oong/ook，通常變ong/ok｜iong/iok）、宕攝合口（wong/wok）、江攝（ong/ok｜iong/iok）有對立時，作爲`oo`使用。
- 「ᅭ」：平時可作`jeon`使用。
- 「ᅨ」「ᅰ」：後備用 
- 韻尾爲`u`者由終聲實現，見下。

### 終聲

<img src="pic\jhg3.png"/>

青色者於實際環境或不能正常組合。

- 「ᆼ」爲鼻化音，粵語或無獨立音位。
- 「ᆯ」可作兒化音。
- `u`不用中聲而用終聲表達是考慮到諺文本身結構同易讀易寫性。

### 聲調

- 【1/4】平聲、上陰入、上陽入不額外標註，如「𡁷（doeng1｜뎡）」
- 【2/5】上聲、中入在諺文左邊標兩點「〯」，如「領（ling5｜링〯ﾞ）」
- 【3/6】去聲、下陰入、下陽入在諺文左邊標一點「〮」，如「趙（zhiu6｜ᅈᅵᆺ〮ﾞ）」
- 陽聲韻再加一個濁音符「ﾞ」即可，如「咬嚼（ngeu5 ziu6｜얏〯ﾞ짓〮ﾞ）」
- 不分高平、高降，特殊調

### 特殊

單獨成韻可直接單獨寫，`m`、`n`、`ng`即爲「ﾱ」「ﾤ」「ﾷ」，`hng`、`hm`	等無元音則省略中間音，即「ᄒᅠᆼ」「ᄒᅠᆷ」（不能正確顯示，）
