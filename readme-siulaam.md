# 小欖話簡介

順德方言主要分佈於順德同其周邊，如中山，南海，番禺等。

順德方面（順德北小片）主要可分爲四種：

1. 以大良爲中心，同周邊容奇、桂州、倫敎、勒流、杏壇爲一類；
2. 以陳村爲中心，同周邊北滘、樂從爲一類；
3. 以龍江爲中心，同周邊龍山、里海爲一類；
4. 以均安爲一類。

中山方面（順德南小片）多爲順德移民，其方言可分爲兩種：

1. 以小欖爲中心，同周邊海洲、東昇、東鳳爲一類；
2. 遠離小欖竝帶有一定蜑家特徵爲一類。

本文介紹小欖話，爲順德南小片之代表，主要介紹其音韻特徵。順德方言概述可見[`readme-dialect.md`](https://github.com/HoengSaan/rime-gukwan/blob/main/readme-dialect.md)。

## 字表

在[Jyutdict](https://jyutdict.org/)有小欖話字表（通表・粵表），可自行查閱。以下爲收錄說明：

- 聲母：不分泥來全讀來，泥母記`l`；鎮區同周邊多數不分透曉，外圍多數分，透母記`t`。
- 韻母：老派無`ak`，全讀`aak`，中新派混讀，部份地方分，本表跟老派作`aak`。師韻老派爲`yu`，中新派受廣州影響部份混（尤其讀書音），部份地區本身就讀`i`，本表作`yu`。`om/op`爲強行高擧，實際中老派多數僅以白讀保留，常見有「柑」「甘」「盒」「鴿」等。字表中有收錄少數棺材白讀。
- 聲調：本身陰去同陽平合，讀陽平，此現象亦影響到部份上陰入口語調，如「哭」「㞘」「捉」等。經調查，即使老派亦不是全讀陽平，而是混讀（通常只有陰上變陽平，僅有極少數反例）。表中全記作陰去【3】。至於上陰入口語調，由於讀作【7】者過於常見，無區別不記；而讀作【3】者應記盡記。變調只記單獨變調者。

- 同廣州話相似之處
  - 古濁聲現今爲陽聲調，鼻音流音近音以外清化：平上有氣（4・5），去入無氣（6）
  - 非敷奉母讀`f`，但有重脣白讀`p`。古微母讀`m`。
  - 不分平翹，古知莊章組`zh/ch/sh`同精組`z/c/s`合流。
  - 日母消失，同疑母細音俱讀`j`。
  - 不分來孃，全讀來母`l`，即「南」`naam4`讀作「藍」`laam4`、「農」`nung4`讀作「龍」`lung4`。
  - 匣母（含少數云母合口字）部份字：即今時廣州話聲母`w`同韻腹`u/o`結合者，順德方言聲母本爲`f`。但小欖話已大幅向廣州貼近，唯「胡」「戶」「護」三小韻混，「胡」多讀作`wu`，「戶」「護」則多讀作`fu`。「垣」小韻讀作`fun`。其他則同廣州一樣讀`w`。
  - 曉匣喻細音字已全面廣州化，讀`j`而非`h`，如「爺」`je4`「賢」`jin4`「雨」`jy5`「藥」`joek6`「欣」`jan1`。不少順德方言亦有此現象。
  - 見組有同廣州話基本一致之弱化現象。
  - 影母讀`q/j/w`，根據韻母而變。
  - 喻母開口讀`j`，合口一般讀`w`。
  - 曉匣母主要讀`h/j/w`，根據韻母而變。
  - `hu(hw)`則變爲`f`，如「庫」`fu3`「訓」`fan3`「忽」`fat1`「闊」`fut3`。
- 同廣州話相異之處
  - 細音區分零聲母`q`同`j`（白讀）：如「應」`ing3/ing1`「煙」`in1/en1`等。
  - 多數人不分透曉，將透母讀作`h`，或混讀`t/h`：如「偸」`tau1`讀`hau1`，「聽」`ting1`讀`hing1`。順德方言中龍江一類亦有類似現象。
  - 塞化/送氣/不送氣，以擦音塞化爲多，其他則較順德話少。
    - 擦音變塞擦，如「剩^白讀^」`zing6`「嘥（浪費）」`caai1`「贖」`zuk6`「煠（水煮）^白讀^」`zep6`「塞^白讀^」`zaak1/zak1`「柿」`zy2`等
    - 無氣字送氣，如「嚿（塊）」`kau4/kau6`「爭（差）^白讀^」`caang1` 「勁（極其之）^白讀^」`king6`「彫^白讀^」`tiu1`
    - 塞擦變擦音，如「齒^白讀^」`si2`「處^白讀^」`syu3`
  - 聲母脫落：「拎^異讀^」`ing1`「搦^異讀^」`ik1`「邇（近稱）」`ei1`
  - 不成規模白讀：「玩」`faan2`「嫌」`him4`「凝・迎」讀`nging4`「睏」`fan3`等，不勝枚擧。
- 補充：`gu(-/i/n/t)`同`ku(-/i/n/t)`除少數例外（擬聲擬態詞）皆讀作圓脣，故省去`w`。

`e`單用時爲[ɛ]，其他則爲[e]（除`eu`部份人士讀[ieu]）。`oe`則較廣州高，讀[ø]。`oen/oet`僅用於白讀。

雖然周邊方言保留有介音，但小欖話同廣州話一樣，小欖話介音已基本消失。（除`j`同`w`，粵語中一般被視爲聲母）裂化程度同其他順德方言類似，不及廣州方言。

小欖話亦完好保留中古三個陽聲韻`m/n/ng`同三個入聲韻`p/t/k`。由於小欖話之中，兩個脣音不能同時出現，所以幫系`m/p`會變成`n/t`。

小欖話有因吞音而造成的長音，見【變調】。

- 通攝內無區分，皆讀`ung`
- 江攝・宕攝一等字讀`ong`，白讀爲`oeng`；江攝照系字・宕攝三等字讀`oeng`
- 止攝
  - 師韻字（即精莊組開口三等字）讀作`y`而非`i`，如「資」`zy1`「師」`sy1`「字」`zy6`「此」`cy2`等。照知組白讀亦有讀作`y`。
  - 見組開口三等字未裂化，如「其」`ki4`「起」`hi2`「機」`gi1`「紀」`gi3`等。但「企」多讀`kei2`。
  - 幫系開合口字、端組來孃開口字皆裂化爲`ei`，如「被」`bei6`「理」`lei5`「卑」`bei1`「你」`lei5`等。有`ai`白讀。
  - 銳音合口字變化爲`eoy`，見系以合口字則變化爲`wai`
- 遇攝
  - 莊組全部讀`o`，如「所」`so2`「阻」`zo2`「初」`co1`「梳」`so1`
  - 一等字除見系以母之外・合口三等字除明母之外，從`u`裂化爲`au`而非`ou`，如「度」`dau6`「無」`mau4`「做（作）」`zau6`「粗」`cau1`等。而疑母則獨立成韻，讀`ng`。
  - 三等字之中，精組來孃母發生裂化讀`eoy`，其他仍讀`yu`，如「句」`gy3`「除」`cy4`「舒」`sy1`「許」`hyu2`
- 蟹攝
  - 開口一等字除泰韻端系讀`aai`之外讀`oi`，二等字讀`(w)aai`，開口三四等字讀`ai`，合口三四等鈍音讀`(w)ai`，銳音以讀`eoy`。
  - 合口一等字讀`ui`，如廣州話讀`eoy`者，小欖話則多讀作`yu`：
    - 端組、來母同心母發生元音融合讀`yu`，如「對」`dy3`「腿」`ty2`「堆」`dy1`「碎」`sy3`等。
    - 精組（除心母）則同廣州一樣低化讀`eoy`，如「最」`zeoy3`「罪」`zeoy6`「催」`ceoy1`等。
- 臻攝
  - 開口有`an/eon`兩種，規律不明，但分佈同廣州一致。
  - 合口一等字之中，見系讀`wan`，幫系讀`un`，端透定來母讀`eon`，精組同泥母讀`yun`。
  - 合口三等字之中，文韻讀`(w)an`，而眞韻之中，銳音讀`eon`，見系以讀`wan`。.
- 山攝
  - 開口一等字之中，幫系見系讀`on`，其他讀`aan`。
  - 開口二等字讀`aan`，山韻有`en`白讀。
  - 開口三四等字讀`in`，四等字有`en`白讀。同廣州方言一樣，「單」爲音旁者，本應讀`in`者讀成`im`。
  - 合口一等字之中，鈍音讀`un`，銳音讀`yun`
  - 合口二等字讀`(w)aan`，刪韻有`en`白讀。
  - 合口三等字元韻幫系讀`aan`，其他讀`yun`。
  - 合口四等字讀`yun`。
- 效攝
  - 一等字發生元音融合，讀`o`，如「高」`go1`「刀」`do1`「毛」`mo4`「草」`co2`。有`au`白讀。
  - 二等字讀`aau`，有`eu`白讀。
  - 三四等字讀`iu`，有`eu`白讀。
- 果攝一等字讀`o`，有`oe`白讀。開口三等字發生元音融合，讀`e`。合口三等字發生元音融合，讀`oe`
- 假攝二等字讀`aa`，三等字發生元音融合，讀`e`，但「也」仍讀`jaa`。
- 梗攝二等字讀`aang`，有`ang`白讀。三四等字讀`ing`，有`eng`白讀。
- 曾攝一等字讀`ang`，有`ing`白讀。三等字讀`ing`。
- 流攝讀`au`。
- 深攝讀`am`，幫組讀`an`。
- 咸攝
  - 一等見系字仍讀`om/op`，如「鴿」`gop3`「合」`hop6`「敢」`gom2`「柑」`gom1`等。但實際情況同方言記錄有差別，多數中年人實際讀`am/ap`，僅部份字作爲白讀保留，以「柑」`gom1`最常見。
  - 一等銳音讀`aam`，但「蠶」有`com4`白讀。幫系讀`aan`。
  - 二等字讀`aam`，幫系讀`aan`。有`em`白讀。
  - 開口三等字，四等字後讀`im`，幫系讀`in`。有`em`白讀。
  - 合口三等字讀`aam`，幫系讀`aan`。
- 多數不分`ak`同`aak`，混讀：如「勒」`lak6`讀`laak6`，「得」`dak1`讀`daak1`。老派完全不分，只有`aak`，無`ak`。
- 白讀
  - 短a變長a（aa）：「第」`daai6`「仔」`zaai2*`「洗」`saai2*`；「某・畝」`maau5`「謀・繆・謬」`maau4`「貿・茂」`maau6`；「侄・姪」`zaat2`「甩」`laat1*`「罰」`faat6`
  - 止攝照知組白讀：「是」`sy6`「馳」`cy4`，「時」`sy4`「蒔」`sy6`等。
  - 止攝`ei`：「祕」`bai3`「姊」`zei2*/ze2*`「仔」`zei2*`「批」`pei1`等。
  - 梗攝`eng`：「稷（穀物）」`zek3`「績・積（地名）」`zek3`「織」`zek1`「鯪」`leng4`「瀝」`lek6`「即」`zek1/ze1`「鼎」`deng2`「蠅」`jeng4`（其他同廣州話同）
  - 咸攝`em/ep`：如「鹹」`hem4`「斬」`zem2`「減」`gem2`「插」`cep3`「餡」`hem2*`等。
  - 效攝`eu`：如「貓」`meu1`「焦」`zeu1`「炮」`peu3`「餃」`geu2`「敲」`heu1`等。
  - 效攝`au`：如「老」`lau5*2`「佬」`lau5*2`「桃」`tau4`。
  - 山攝`en/et`：如「鏟」`cen2`「間」`gen1`「扁」`ben2`「挖」`wet1`「莧」`hen6`等。
  - `o`白讀讀`oe`：「螺」`loe4`「墮」`doe6`「朵」`doe2`「多」`doe1`「糯」`loe6`等。
  - 不成規模白讀：「鏟」`coen2`「閂」`soen1/soet1*`「就」`zou6/a6`「閃」`saam2`「疊」`daap6`「掩」`om2`「蠶」`com4/com2'`「貶」`ben2`「拿｛推測｝」`lo2`「頂」`doeng2`「酗」`ho3`「壅」`ung1`「挐」`laa1`「嚼」`ziu6/zeu6`「側・則（文讀爲`zik1`）」`zaak1`「裝」`zoeng1`等，不勝枚擧。

  - 下陰入韻母：`op→ap(aap) [ɔː]`，`ot [ɔː]`，`ok [ɔː]`，`oet [øː]`，`oek [øː]`，`ip [iː]`，`it [iː]`，`ep [eː]`，`et [eː]` ，`ek [eː]`，`aap [aː]`，`aat [aː]`，`aak [aː]`
  - 小欖話現今上陰入或爲廣州話借調，陰入分化前或爲「4」，本人將其標爲【7】，此聲調多數出現於口語字或地名。
    - 如「福興」`fuk1* hing1`，「吉屋」`gat1* uk3	`「出」`ceot1*/cyt1*/cot1*`，「棘（頭、腦、頭痛）」`kik1*`「踢（鞋）`hek1*/tek1*`等。
    - 部份上陰入字口語則讀作下陰入或上陽入（陽平），如「屋企」`uk3/uk4 kei2`「哭」`huk3「捉」`zuk3/zuk4`等，
- 高平高降條件未明。

## 變調・口語調

### 陰平類

#### 【1*】【7】（44/4）：

- 變調：前字爲陰平・上陰入字，而後字爲變調【2'】字時，前字則讀【1\*】，後字爲超高陰平【1'】或陰平【1】，高於文中其他陰平字。《中山方言誌》將其標爲33，但實際此字調值較陰去稍高，本人認爲「44/4」更合適。
  - **1 + 2' => 1*/1 + 1'**：如「窗簾」`coeng1* lim1'`「蕉蕾」`ziu1* leoy1'`「番薯」`faan1* sy1'`。
  - **1 + 1 + 2' => 1 + 1* + 1'**：如「開窗簾」`hoi1 coeng1* lim1'`「煨番薯」`wui1 faan1* sy1'`。

- 變調：若前後字皆爲陰平，有部份詞前字會讀作【1*】，規律未知。
  - **不變**：「溫州」`wan1 zau1`「先生」`sin1 saang1`「徵招」`zing1 ziu1`。
  - **1 + 1 => 1* + 1**：「東昇（地名）」`dung1* sing1`「珍珠」`zan1* zy1`「捉伊人」`zuk3 ji1* jan1`
  - **1 + 1 + 1 => 1 + 1* + 1/1'**：「釘書機」`deng1 sy1* gi1`
  - **上陰入字極少例外**：「北京」`baak1* ging1`「福興」`fuk1* hing1`「色精」`sik1* zing1`

- 文法：表達過去・**<u>完成</u>**。元本形態或爲「動詞」+「咗」+「先」。小欖話表達過去・完成，動詞不少要發生變調。亦有讀作陰平【1】者。
  - 如「熄掣（關了開關）」`sik1* zai3`，「食喇（食了）」`sik1* la2`「去街（出了門）」`hy1* gaai1`
  - 例句：「食飯就同你出街（喫了飯就和你出門）」`sik1* faan6 a6 hung4 lei5 ceot1* gaai1`
  - 變調者發作長音，長音符暫作「h」，加在韻尾後。如「出去（出了門）」`cyth1* hy4/hy3`。

- 文法：表達意向。【長音】是「吞音」造成，元本形態爲「動詞」+「一」+「動詞」。發生「吞音」後，中間「一」消失，首字變【1*】或【2\*】。

  - 如「餵一餵魚」在小欖話則讀作「餵餵魚」`wai1* wai3 jy2'`或；如「食一食嘢」則「食食嘢」`sik1* sik6 je5`
  - 變調者多發作長音，長音符暫作「h」，加在韻尾後。

#### 【1'】超高陰平：見上。

#### 【1】陰平變調（55）：

- 廣州話本身變調爲陰平者，小欖話則爲【1】或【2'】。
  - 如「自己」`zy6 gi1`（可不變）「阿姨」`aa3 ji1`「返屋企」`faan1 uk3 kei2`

- 高平高降條件未明。

### 陰上類

#### 【2*】陰上變調一類：

- 部份人無區分【2】同【2*】。
- 變調：同廣州話，但並非完全一致。基本只發生於非陽平字。
  - 「坐月」`co5 jyut2*`「雀」`zoek2*`「男仔」`laam4 zaai2*`，詳細見「2'」段。
  - 別意：「伯父（男性長者）」`baak3 fu2*`「伯父（親戚）」`baak3 fu6`
- 文法：表達<u>**過去**</u>・完成。爲動詞後「咗（zo2）」影響，變調後「咗」多數被吞，但仍可接「先」。（接「咗」時多不變調）
  - 如「食飯（喫了飯）」`sik2* faan6`「唔見先（不見了）」`m4 gen2* sin1`
  - 元本完整形態應爲「動詞」+「咗」。
  - 變調者發作長音，長音符暫作「h」，加在韻尾後。
- 文法：表達意向，見**陰平類**。

#### 【2'】陰上變調二類：

- 廣州話陽平變調爲陰上者，小欖話則多變調作【2'】
  - 連續變：「芥蘭」`gaai3 laan2'`「舖頭」`pau3 hau2'`「臘腸」`laap6 coeng2'`「女人」`leoy5 jan2'`「還錢」`wen4 cin2'`「幾時」`gi2 si2'`「甜甜地」`tim4 tim2' dei2`
  - 單字變：「錢」`cin2'`「錘」`ceoy2'`「刨」`peu2'`「糖（糖果）」`tong2'`（作詞變調爲「2*」）「鵝」`ngo2'`「魚」`jyu2'`「桃」`ho2'`「蠶」`com2'`「壺」`wu2'`等。，類似廣州話，如「文（單位）」`man1`「朦」`mung1`「瞄」`miu1`
  - 常見例外：

    - 「廚」（規律不明）：「廚房」`cy2* fong2'`

  - 常見例外（小稱）：變「2*」，如「肥」「王」「胡」「黃」等

## 文法

文法同廣州話非常類似。

### 人稱代詞

複數加「哋」`dei5/lei5`

- 第一身：「我」`ngo5/ngoi5`
- 第二身：「你」`lei5`
- 第三身：「渠」`hy5/jy5`

### 指示代詞

小欖話同廣州話不同，仍保留有一定程度上中遠指之區別。

| 近指             | 中遠指             | 遠指         | 不定指                 |
| ---------------- | ------------------ | ------------ | ---------------------- |
| 「邇」`lei1/ei1` | 「個」`go2*/goi2*` | 「那」`lau6` | 「邊」`bin1/min1/ben1` |

- 組詞例
  - 「邇個」`~ go4/goi4/go3`
  - 「邇的」`~ di1`
  - 「邇處」`~ sy2*`
  - 「邇度」`~ dau6`
  - 「邇邊」`~ bin1/min1/ben1`

- 「那」亦有另外的意思。
  - 「渠在邊？——渠在～邊。」（他在哪裏？——他在那^遠^裏）
  - 「你在邊？——我在～邊。」（你在哪裏？——我在另外一邊）


## 文灋用詞

- 介詞
  - 地點「在」`coi2`：多用於生物
  - 地點「喺」`hai2`：多用於非生物
  - 地點「gwaa1」`gwaa1`：只能接動詞
    - 「誃歸好即刻丟的肉～雪櫃度」（回家之後記得立刻將肉放到冰箱裏。）
